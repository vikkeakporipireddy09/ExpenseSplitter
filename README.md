# ExpenseSplitter
 A simple web app to split group expenses and track who owes whom. Add members, log expenses, and view real-time balances. Built using HTML, CSS, and JavaScript — perfect for trips, roommates, or shared events.



Expense Splitter – SplitWise Clone Lite
Expense Splitter is a lightweight web application that helps groups of friends or roommates track shared expenses and see who owes whom — inspired by SplitWise.

Features
Add members to the group

Add expenses with amount, description, and payer

Automatically splits the expense equally among all members

Shows who owes whom and how much

Tech Stack
HTML

CSS

JavaScript

How to Use
Add names of all members in the group.

Add any expense with a short description, amount, and who paid.

The app calculates the fair split and displays balances.

Live Preview:
You can run it locally or deploy it using GitHub Pages, Netlify, or Vercel.

Use Case:
Perfect for managing group trips, flatmate expenses, shared meals, or any group spending situation.
